
# Nghiên cứu Ứng dụng Công nghệ Phân tán trong Quản lý An sinh Xã hội

## Giới thiệu
Dự án này tập trung nghiên cứu và phát triển giải pháp lưu trữ, xử lý dữ liệu an sinh xã hội quy mô lớn. Hệ thống sử dụng kiến trúc Microservices, cơ sở dữ liệu phân tán và công nghệ Big Data để đảm bảo tính toàn vẹn và hiệu năng.

## Mục tiêu
- Xây dựng mô hình dữ liệu chuẩn hóa cho hồ sơ an sinh xã hội.
- Tối ưu hóa tốc độ truy vấn và đồng bộ dữ liệu giữa các node.
- Đảm bảo tính bảo mật và minh bạch trong quy trình xét duyệt.

## Phương pháp
- **Công nghệ:** Java Spring Boot, Apache Spark, MongoDB, Oracle.
- **Tiếp cận:** Sử dụng thuật toán đồng thuận (Consensus) để xác thực dữ liệu đa điểm.

## Cấu trúc thư mục
- `data/`: Dữ liệu mẫu (đã được ẩn danh).
- `src/`: Mã nguồn dự án (Backend & Frontend).
- `docs/`: Tài liệu thiết kế hệ thống và báo cáo.
- `results/`: Kết quả thực nghiệm và biểu đồ hiệu năng.
